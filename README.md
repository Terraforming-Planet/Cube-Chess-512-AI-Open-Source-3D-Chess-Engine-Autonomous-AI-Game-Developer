# Cube-Chess-512-AI-Open-Source-3D-Chess-Engine-Autonomous-AI-Game-Developer
Open-source projekt profesjonalnej gry szachowej 3D 8×8×8 (512 pól) rozwijanej przez autonomicznych agentów AI. Projekt obejmuje silnik gry, sztuczną inteligencję, grafikę 3D, multiplayer, testy, dokumentację i architekturę gotową do publikacji oraz dalszego rozwoju.
